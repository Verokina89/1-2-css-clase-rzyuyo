Selectores, modelo de cajas, propiedades de texto y flexbox

Los selectores en CSS son las reglas que usamos para seleccionar los elementos que queremos modificar. Hay muchos tipos de selectores, cada uno con su propio propósito.

Selectores

Los selectores básicos se utilizan para seleccionar elementos por su nombre, clase o ID.

  • Por nombre o elemento: Se utiliza para seleccionar todos los elementos con un nombre determinado.
  • Por clase: Se utiliza para seleccionar todos los elementos con una clase determinada.
  • Por ID: Se utiliza para seleccionar un elemento con un ID determinado.
  • Por atributo: Se utiliza para seleccionar elementos con un atributo determinado.

Modelo de cajas

El modelo de cajas en CSS es un modelo conceptual que describe cómo se representan los elementos HTML en la pantalla. El modelo de cajas se divide en cuatro partes:

  • Contenido: Es el contenido real del elemento, como el texto o la imagen.
  • Margen: Es el espacio entre el contenido y el borde de la caja.
  • Relleno: Es el espacio entre el contenido y el borde del relleno.
  • Borde: Es el borde que rodea la caja.

Las propiedades del modelo de cajas se utilizan para controlar el tamaño y el aspecto de las cajas. Algunas de las propiedades más comunes son:

  • width: Establece el ancho de la caja.
  • height: Establece el alto de la caja.
  • margin: Establece el margen de la caja.
  • padding: Establece el relleno de la caja.
  • border: Establece el borde de la caja.

Flexbox

Flexbox es un módulo CSS que permite crear diseños flexibles y adaptables. Flexbox utiliza un sistema de ejes para colocar los elementos de una página web.

Las propiedades de flexbox se utilizan para controlar el comportamiento de los elementos en un contenedor flexbox. Algunas de las propiedades más comunes son:

  • display: Establece el tipo de contenedor flexbox.
  • flex-direction: Establece la dirección del eje principal.
  • justify-content: Establece el alineamiento de los elementos a lo largo del eje principal.
  • align-items: Establece el alineamiento de los elementos a lo largo del eje transversal.
  • flex-wrap: Establece si los elementos se envuelven o no a lo largo del eje principal.
